'Map Displacement Filter #2' tags: - canvas - filter - HTML5 lab - image - interactive - manipulation - map displacement - pixel status: publish type: post published: true --- And here comes the other Map Displacement filter demo. To apply the filter, click anywhere on the picture, move the mouse around and release the click... tadaaa! Be gentle with Mona' thought. The filter isn't real time applied because of performance issues. It takes about half a second to calculate the displacement, where it would have to take 10 times less to consider doing it in real time. But it could be done, I just didn't think it was worth the effort for this demo. Another note on the performance issue : the amount of time it takes is mainly due to the canvas pixel array accessing stuff. And apparently, things get a lot better with Firefox 4, and I wouldn't wonder if Chrome wasn't putting some effort into this too... which means we'll see more and more of this kind of effects \o/
